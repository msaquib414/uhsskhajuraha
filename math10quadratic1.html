द्विघात समीकरण MCQ टेस्ट

1. यदि द्विघात समीकरण x² + 5x + 6 = 0 है, तो इसके मूल क्या हैं?

सही उत्तर: a) -2, -3

2. द्विघात समीकरण 2x² - 7x + 3 = 0 का विविक्तकर (discriminant) क्या है?

सही उत्तर: b) 25

3. यदि द्विघात समीकरण के मूल समान हों, तो विविक्तकर का मान क्या होता है?

सही उत्तर: c) = 0

4. समीकरण x² - 9x + 20 = 0 के मूलों का योग क्या होगा?

सही उत्तर: b) 9

5. यदि एक मूल 2 और मूलों का योग 5 है, तो दूसरे मूल का मान क्या होगा?

सही उत्तर: a) 3

6. द्विघात समीकरण 3x² + 9x + 6 = 0 के मूलों का गुणनफल क्या होगा?

सही उत्तर: a) 2

7. निम्नलिखित में से कौन सा द्विघात समीकरण नहीं है?

सही उत्तर: c) x + 1/x = x²

8. यदि द्विघात समीकरण x² - kx + 4 = 0 के मूल वास्तविक और समान हैं, तो k का मान क्या होगा?

सही उत्तर: a) ±4

9. द्विघात समीकरण x² - 7x + 12 = 0 के मूल क्या होंगे?

सही उत्तर: a) 3, 4

10. समीकरण (x-3)² = 49 को हल करने पर x का मान क्या होगा?

सही उत्तर: a) 10, -4

11. समीकरण x² - 5x = 0 के मूल हैं:

सही उत्तर: a) 0, 5

12. यदि द्विघात समीकरण x² + bx + c = 0 का एक मूल -1 है, तो निम्नलिखित में से क्या सत्य है?

सही उत्तर: b) 1 + b + c = 0

13. द्विघात समीकरण 9x² - 6x + 1 = 0 के मूलों की प्रकृति कैसी होगी?

सही उत्तर: a) वास्तविक और बराबर

14. यदि द्विघात समीकरण x² - 4x + k = 0 का एक मूल 4 है, तो k का मान क्या होगा?

सही उत्तर: a) 0

15. द्विघात समीकरण 2x² + 3x + 5 = 0 के मूल कैसे होंगे?

सही उत्तर: a) काल्पनिक

16. समीकरण √(2x+9) = x - 3 को हल करने पर x का मान क्या होगा?

सही उत्तर: b) 7

17. यदि α और β समीकरण 2x² + 5x - 12 = 0 के मूल हैं, तो α + β का मान क्या होगा?

सही उत्तर: a) -5/2

18. द्विघात समीकरण x² + 6x + 9 = 0 के मूलों का विविक्तकर क्या होगा?

सही उत्तर: a) 0

19. यदि द्विघात समीकरण 3x² - kx + 5 = 0 का एक मूल 1 है, तो k का मान क्या होगा?

सही उत्तर: a) 8

20. समीकरण x² + 2x - 3 = 0 के मूल कैसे होंगे?

सही उत्तर: a) वास्तविक और असमान

21. द्विघात समीकरण 4x² - 3x + 2 = 0 के मूलों का योग क्या होगा?

सही उत्तर: a) 3/4

22. यदि द्विघात समीकरण 3x² + kx + 2 = 0 के मूल एक-दूसरे के व्युत्क्रम हैं, तो k का मान क्या होगा?

सही उत्तर: a) -3

सही उत्तर: a) वास्तविक और असमान

24. यदि द्विघात समीकरण ax² + bx + c = 0 का विविक्तकर 0 है, तो मूल कैसे होंगे?
