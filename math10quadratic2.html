द्विघात समीकरण MCQ टेस्ट - नया सेट

1. द्विघात समीकरण का मानक रूप क्या है?

सही उत्तर: b) ax² + bx + c = 0

2. यदि द्विघात समीकरण x² - 6x + 9 = 0 है, तो इसके मूल हैं:

सही उत्तर: a) 3, 3

3. विविक्तकर (discriminant) का सूत्र क्या है?

सही उत्तर: b) b² - 4ac

4. यदि विविक्तकर > 0 है, तो मूलों की प्रकृति कैसी होगी?

सही उत्तर: c) वास्तविक और असमान

5. समीकरण 2x² + 4x + 1 = 0 के मूलों का गुणनफल क्या होगा?

सही उत्तर: b) 1/2

6. यदि α और β द्विघात समीकरण के मूल हैं, तो α + β किसके बराबर है?

सही उत्तर: d) -b/a

7. समीकरण x² - 5x + 6 = 0 को गुणनखंड विधि से हल करें, मूल क्या होंगे?

सही उत्तर: c) 2, 3

8. यदि एक द्विघात समीकरण का विविक्तकर ऋणात्मक है, तो कितने वास्तविक मूल होंगे?

सही उत्तर: c) 0

9. समीकरण x² - 16 = 0 के मूल क्या हैं?

सही उत्तर: a) 4, -4

10. यदि द्विघात समीकरण के मूल वास्तविक और बराबर हैं, तो विविक्तकर का मान क्या होगा?

सही उत्तर: a) 0

11. समीकरण (x + 3)² = 0 के मूल हैं:

सही उत्तर: a) -3, -3

12. यदि द्विघात समीकरण के मूल 4 और -2 हैं, तो समीकरण क्या होगा?

सही उत्तर: a) x² - 2x - 8 = 0

13. द्विघात सूत्र (Quadratic Formula) क्या है?

सही उत्तर: a) x = [-b ± √(b² - 4ac)] / 2a

14. यदि समीकरण x² + px + q = 0 के मूल समान हैं, तो p² किसके बराबर है?

सही उत्तर: a) 4q

15. समीकरण 3x² - 5x + 2 = 0 के मूलों का योग क्या होगा?

सही उत्तर: a) 5/3

16. यदि समीकरण x² + kx + k = 0 के मूल वास्तविक और भिन्न हैं, तो k के लिए क्या शर्त है?

सही उत्तर: a) k < 0 या k > 4

17. यदि α, β समीकरण ax² + bx + c = 0 के मूल हैं, तो αβ किसके बराबर है?

सही उत्तर: a) c/a

18. समीकरण x² - 4x + 5 = 0 का विविक्तकर क्या होगा?

सही उत्तर: a) -4

19. यदि समीकरण x² - 2x + m = 0 का एक मूल 0 है, तो m का मान क्या होगा?

सही उत्तर: a) 0

20. समीकरण x² + 7x + 10 = 0 के मूल कैसे होंगे?

सही उत्तर: a) वास्तविक और असमान

21. द्विघात समीकरण 6x² + x - 2 = 0 के मूलों का योग क्या होगा?

सही उत्तर: a) -1/6

22. यदि द्विघात समीकरण 2x² - 8 = 0 है, तो x का मान क्या होगा?

सही उत्तर: a) ±2

23. समीकरण x² - 2x + 3 = 0 के मूलों की प्रकृति क्या होगी?

सही उत्तर: a) काल्पनिक

24. यदि द्विघात समीकरण के मूलों का योग और गुणनफल क्रमशः 3 और 2 हैं, तो समीकरण क्या होगा?

सही उत्तर: a) x² - 3x + 2 = 0

25. समीकरण 4x² - 12x + 9 = 0 के मूलों की प्रकृति क्या होगी?

सही उत्तर: a) वास्तविक और बराबर
